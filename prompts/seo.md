# SEO Prompts — Petacular

---

## Prompt 1: Homepage Meta Description
```
Write a Google meta description for the homepage of Petacular, a pet products business.
Max 155 characters.
Include the primary keyword and a value proposition.
```

## Prompt 2: Service Page Meta Description
```
Write a meta description for a service page about [specific service] for Petacular.
Max 155 characters. Include keyword and CTA.
```

## Prompt 3: Keyword Research Brief
```
List 15 SEO keywords a potential customer might search when looking for pet products services.
Include: 5 short-tail keywords, 5 medium-tail, 5 local/long-tail (include [suburb/city]).
Format as a table: Keyword | Type | Search Intent.
```

## Prompt 4: Location Page Copy
```
Write 200 words of copy for a local landing page targeting [suburb] residents who need pet products services.
Include the suburb name naturally 3-4 times.
Mention Petacular and include a CTA to request a quote.
```

## Prompt 5: Title Tags
```
Write 5 title tag options for the homepage of Petacular.
Each must be under 60 characters and include the primary keyword.
```
